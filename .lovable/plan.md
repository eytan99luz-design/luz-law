

# אתר דף נחיתה - עו"ד איתן לוז

- **ערכת צבעים**: רקע כהה (שחור/אפור כהה) עם אלמנטים בזהב — מראה יוקרתי וקלאסי
- **טיפוגרפיה**: פונטים אלגנטיים בעברית ובאנגלית
- **כיוון**: RTL לעברית, LTR לאנגלית, עם כפתור החלפת שפה בהדר

## מבנה הדף (One-Page עם סקשנים)
1. **Hero Section** — שם, תואר, תמונת פרופיל עם אנימציית כניסה, כפתור "צור קשר"
2. **אודות** — פסקת הצגה עם תמונה, אנימציית fade-in בגלילה
3. **תחומי עיסוק** — כרטיסים עם אייקונים: דיני חוזים, נדל"ן, דיני תאגידים, ייעוץ לעמותות, ליטיגציה אזרחית, גישור ובוררות
4. **למה לבחור בי** — 3-4 יתרונות עם אייקונים ואנימציות
5. **טופס יצירת קשר** — שם, טלפון, מייל, הודעה → שליחה למייל eytanluz.law@gmail.com (דרך Edge Function)
6. **פוטר** — כתובת, טלפון, מייל, מפה (אופציונלי)

## אנימציות
- אנימציות כניסה בגלילה (fade-in, slide-up) לכל סקשן
- אפקט hover על כרטיסי תחומי העיסוק
- אנימציית scale על כפתורים

## דו-לשוניות (עברית + אנגלית)
- כפתור החלפת שפה בהדר
- כל הטקסטים מנוהלים דרך קובץ תרגום
- החלפת כיוון RTL/LTR אוטומטית

## דף ניהול (Admin Panel)
- **התחברות**: מייל וסיסמה דרך Supabase Auth (Lovable Cloud)
- **עריכת תוכן**: טקסטים, תמונות, תחומי עיסוק — נשמרים ב-DB
- **ניהול תמונות**: העלאת ושינוי תמונות דרך Supabase Storage
- גישה דרך נתיב `/admin`

## Backend (Lovable Cloud)
- **Database**: טבלה לתוכן האתר (טקסטים ותמונות לכל סקשן בשתי שפות)
- **Auth**: התחברות אדמין עם מייל וסיסמה
- **Edge Function**: שליחת טופס יצירת קשר למייל
- **Storage**: באקט לתמונות האתר
